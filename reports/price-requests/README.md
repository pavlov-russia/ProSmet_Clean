# Price Input Requests

AI-сотрудники создают здесь запросы, когда реальный прайс становится блокером:

```text
reports/price-requests/PRICE-REQUEST-YYYYMMDD-TASK-ID.json
```

Формат: `docs/contracts/PriceInputRequest.md`.

До появления реального прайса implementation работает только на synthetic fixtures.
