# handoffs

Здесь фиксируются handoff-документы между AI-сотрудниками разработки.

Формат:

```text
Handoff_XX_short-name.md
```

Обязательные поля:

- тип документа;
- дата;
- автор / роль;
- родительский документ;
- родительская задача;
- цель ветки;
- зависимости;
- что сделано;
- что осталось;
- quality gates;
- следующий исполнитель.
