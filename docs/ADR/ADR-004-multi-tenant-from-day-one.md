# ADR-004 · Мультитенантность с первого дня

Дата: 2026-05-19  
Статус: принято

## Контекст

ProSmet обслуживает разные бригады. Утечка лидов, смет, прайсов или переписки между бригадами является критическим провалом.

## Решение

Изоляция данных строится с первого дня через PostgreSQL Row-Level Security. Все чувствительные таблицы получают прямой `tenant_id`.

## Последствия

- Tenant берется только из серверной сессии.
- `tenantId` из клиента не является доверенным.
- Приложение не имеет `BYPASSRLS`.
- RLS и cross-tenant access входят в обязательные тесты.
