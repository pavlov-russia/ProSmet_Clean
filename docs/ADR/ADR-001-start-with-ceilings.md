# ADR-001 · Стартуем с натяжных потолков

Дата: 2026-05-19  
Статус: принято

## Контекст

Долгий горизонт ProSmet — модульная строительная платформа. Но первая реализация должна доказать ценность в одной вертикали с понятными параметрами, продажами и эталонными сметами.

## Решение

Первая вертикаль — натяжные потолки.

## Последствия

- Потолочная предметка живет в `apps/ceiling`.
- Общие сущности живут в `packages/core`.
- Первые пилоты, golden estimates и кастдев ищутся только в потолочной нише.
- Остальные вертикали фиксируются как будущие модули.
