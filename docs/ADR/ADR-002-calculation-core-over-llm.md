# ADR-002 · Расчетное ядро важнее LLM

Дата: 2026-05-19  
Статус: принято

## Контекст

Ошибка в смете стоит денег, доверия и может создать юридический риск. LLM хорошо собирает и объясняет информацию, но плохо подходит как источник цены.

## Решение

Деньги считает только детерминированное расчетное ядро. LLM собирает, нормализует, проверяет пропуски и формирует объяснения вокруг уже рассчитанного результата.

## Последствия

- В AI payload не передаются денежные суммы.
- Каждая смета хранит версию формулы и прайса.
- Golden estimates становятся обязательным regression set.
- Расхождение LLM и формулы решается в пользу формулы.
