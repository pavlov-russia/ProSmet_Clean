# .claude/rules

Здесь размещаются path-scoped rules Claude Code.

Когда использовать:

- правило относится только к `packages/core`;
- правило относится только к `apps/ceiling`;
- правило относится только к миграциям БД;
- правило относится только к AI prompts или PDF templates.

Пример будущих правил:

- `core-boundary.md`;
- `ai-gateway.md`;
- `db-rls.md`;
- `pdf-legal.md`.
