# Orchestrator Workspace

Локальная рабочая зона development-time AIOrchestrator.

- `approvals/` — machine-readable решения архитектора по orchestration cycle.
- `outbox/` — command packets для AI-сотрудников, созданные только после approval.

Эта папка не хранит секреты, реальные ПД или реальные прайсы.
