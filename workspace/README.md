# workspace

Рабочая зона временных артефактов.

Сюда можно складывать:

- черновые схемы;
- временные таблицы;
- промежуточные заметки;
- результаты анализа.
- запросы архитектору в `workspace/architect-inbox/`.

Канонические решения должны переноситься из `workspace/` в `docs/`, `tasks/`, `reports/` или ADR.
