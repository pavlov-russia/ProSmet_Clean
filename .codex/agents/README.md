# .codex/agents

Здесь могут жить профили Codex-агентов, если проект будет использовать локальные agent profiles.

До появления реальных профилей источником истины для Codex является `AGENTS.md`.

Рекомендуемые профили должны соответствовать ролям из `harness/agent-roles.md`, включая senior AI-сметчика по натяжным потолкам.
